Multimedia in HTML

Multimedia in HTML means adding different types of media (like images, audio, video, animations, and interactive content) to a webpage along with text.

Without multimedia: a book with only text.
With multimedia: a movie with sound, visuals, and subtitles.

1. Image (<img>)

The <img> tag is used to embed images. It is self-closing and an inline element.

  • src → Path of image
  • alt → Alternate text
  • width/height → Dimensions

2. Audio (<audio>)

The <audio> tag is used to embed sound in a webpage.

  • src → Path of audio
  • controls → Show play, pause, volume
  • autoplay → Play automatically
  • loop → Repeat audio
  • muted → Start muted

Example:

3. Video (<video>)

The <video> tag is used to embed videos in a webpage.

  • src → Path of video
  • controls → Show video controls
  • autoplay → Play automatically
  • loop → Repeat video
  • muted → Start without sound
  • poster → Thumbnail before play
  • width/height → Size

4. Iframe (<iframe>)

The <iframe> tag is used to embed another webpage inside the current page.

  • src → URL of page
  • target → Where links open (_self, _blank, _top, etc.)
  • frameborder → Show/hide border (deprecated; use CSS instead)
