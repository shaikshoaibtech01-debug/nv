HTML Lists

A list in HTML is a way to group related items together in a structured format using special tags.

Types of Lists in HTML:

  1. Ordered List
  2. Unordered List
  3. Definition/Description List

For list items we use the <li>item</li> tag.

1. Ordered List

An ordered list displays items with numbers, letters, or roman numerals.
For ordered lists, we use the <ol> tag.

Attributes in <ol> tag:

  • type → Defines numbering style (values: "1", "A", "a", "I", "i")
  • start → Sets the starting value
  • reversed → Boolean attribute; shows list in descending order

Examples:

Default (numbers):

  1. Apple
  2. Banana
  3. Mango

Type "A":

  1. Apple
  2. Banana
  3. Mango

Start from 5:

  1. Apple
  2. Banana
  3. Mango

Reversed order:

  1. Apple
  2. Banana
  3. Mango

2. Unordered List

An unordered list displays items with bullets.
For unordered lists, we use the <ul> tag.

Example:

  • Apple
  • Banana
  • Mango

3. Definition/Description List

Used for terms and their descriptions.
<dl> → Description list container
<dt> → Description term
<dd> → Description definition

Anchor Tag (<a>)

The anchor tag in HTML is used to create hyperlinks. It connects one page to another, or one section of a page to another.

Attributes:

  • href → Specifies the URL or file path (e.g. href="https://google.com")
  • target → Specifies where to open the link:
    • _self → same tab (default)
    • _blank → new tab/window
  • title → Shows extra info when you hover over the link

Examples:

Google (same tab)
Google (new tab)
