Marquee Tag

The <marquee> tag is used to create scrolling (moving) text or images inside a webpage. ⚠️ Note: The marquee tag is deprecated in HTML5 (not recommended for modern websites). Instead, use CSS animations.

Difference between Div and Span

Div:

  • Block-level element
  • Starts on new line, takes full width
  • Can set width & height

Span:

  • Inline element
  • Stays in same line
  • For styling small parts of text
