Text Formatting Tags:

Formatting tags are used to change the appearance of text like making it bold, italic, underlined, small, big, marked, etc.

Bold Text: Makes the text bold without giving it extra importance.

Strong Importance: Highlights text as important; browsers render it as bold.

Italic Text: Makes the text italic without emphasizing importance (used for style, like book titles).

Emphasized Text: Emphasizes text; browsers render it in italic and screen readers put stress on it.

Underline: This is underlined text.

Highlighted Text: This is highlighted with a yellow background (like a marker).

Smaller Text: This is smaller than normal text.

Deleted Text: This is deleted or removed (strike-through).

Inserted Text: This is inserted text (displayed as underlined).

Subscript: H2O (water)

Superscript: 102 = 100

Big Text: The big tag increases font size by one level (deprecated in HTML5). Instead, use CSS like font-size.
